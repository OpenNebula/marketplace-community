# Changelog

All notable changes to the Nextcloud All-in-One appliance will be documented in this file.

## [1.0.0-2] - 2026-01-07

### Changed
- Renamed appliance from nextcloud-suse to nextcloud-aio
- Updated container name to nextcloud-aio-mastercontainer (required by Nextcloud All-in-One)
- Added documentation links to appliance description

## [1.0.0-1] - 2025-12-17

### Added
- Initial release of Nextcloud All-in-One appliance
- Docker container: nextcloud/all-in-one:latest
- VNC desktop access
- SSH key authentication
- OpenNebula context integration
- Configurable container parameters
