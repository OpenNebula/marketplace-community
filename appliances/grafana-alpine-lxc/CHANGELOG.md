# Changelog

All notable changes to this appliance will be documented in this file.

## [1.0] - 2026-01-21

### Added
- Initial release of grafana-alpine-lxc
- Grafana Dashboard pre-configured and ready to use
- Alpine Linux 320.aarch64 base
- contextless deployment support
- Optimized for aarch64 architecture
- Compatible with OpenNebula 6.10 and 7.0

### Features
- Service running on port 3000
- Lightweight LXC container
- Edge/IoT device compatible
